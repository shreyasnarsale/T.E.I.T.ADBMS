##  Assignments

### **1. MongoDB Database Operations**

* Create a sample database and collection.
* Perform:

  * Insert (batch insert, validation)
  * Remove documents
  * Update (replacement, modifiers, upserts, multiple updates)
* Execute at least **10 queries** using:

  * `find`, `findOne`, conditionals (`$or`, `$not`)
  * Type-specific, regex, and array queries
  * `$where` and cursor operations (`limit`, `skip`, `sort`)

---

### **2. MapReduce, Aggregation & Indexing**

* Demonstrate **Aggregation Framework**.
* Create and drop different types of **indexes**.
* Use `explain()` to show index performance benefits.
